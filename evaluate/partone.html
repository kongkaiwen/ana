Part One

The first part of my evaluation is generating questions that are suitable for a given conversation. You will see a dialogue between two people and generated questions. You will need to first choose which question is the best, and then for each question check if that question fits the current context.

  • Choose the best question out of all possible questions.
  • For each question is it "reasonable". Does it make sense?
